🏆 Vasileuski Sports API 🏆

Добро пожаловать в API управления спортивными командами и категориями.

📚 Открыть Swagger документацию

📡 Доступные API endpoints:

Категории

GET /api/categories

Получить все категории

GET /api/categories/{id}

Получить категорию по ID

POST /api/categories

Создать новую категорию

Команды

GET /api/teams

Получить все команды

GET /api/teams/{id}

Получить команду по ID

POST /api/teams

Создать новую команду

GET /api/teams/search?term=текст

Поиск команд

Статистика

GET /api/teams/statistics

Получить статистику

GET /api/teams/top/{count}

Топ N команд
